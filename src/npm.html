  • NPM is the Node Package Manager
  • Installed as part of the Node.js installation
  • Very popular tool for managing JavaScript packages
  • A huge library of packages are available at https://www.npmjs.com
Command What it does
npm --version outputs the version of npm
npm list Lists installed packages
npm install typescript Installs a package (TypeScript in this case)
npm install typescript@5.4.5 Installs a specific version of a packages
npm install Installs packages as specified in the package-lock.json file
npm uninstall typescript Un-installs package
