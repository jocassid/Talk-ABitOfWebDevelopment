Bulma (https://bulma.io/)

  • Pure CSS framework (Inevitably, you'll need to write a bit of JavaScript)
    • PRO: Doesn't pull in a huge block of JavaScript
    • PRO: Write JavaScript the way you want to
    • CON: You'll probably wind up writing a bit more JavaScript than you would with something like bootstrap
    • Documentation is pretty good about providing samples where you need to add JavaScript
  • Based on flexbox & grid
  • Define themes using SASS/SCSS
<button class="button is-white">White</button>
<button class="button is-light">Light</button>
<button class="button is-dark">Dark</button>
<button class="button is-black">Black</button>
<button class="button is-text">Text</button>
<button class="button is-ghost">Ghost</button>
