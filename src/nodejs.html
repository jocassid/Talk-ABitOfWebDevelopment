Node.js is a JavaScript interpreter that runs outside a browser

https://nodejs.org

Includes a standard library which covers things such as file I/O, operating system calls, networking, etc.

You can run web applications with Node.js in much the same way as you can with a Python framework like Flask (If you're into that sort of thing)

Even if you're not writing a Node.js application, you're probably using it's package manager (more on that later)
